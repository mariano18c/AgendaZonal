# Historias de Usuario

## Vecino (Usuario Final)
1. **Búsqueda Hiperlocal**: Como vecino, quiero buscar servicios (ej: "Plomero") filtrando por mi zona actual para encontrar ayuda inmediata sin traslados largos.
2. **Confianza y Reseñas**: Como usuario, quiero leer comentarios de otros vecinos y ver fotos de trabajos reales para elegir un proveedor confiable.
3. **Modo Offline**: Como vecino en zona rural, quiero que la app funcione sin internet para consultar teléfonos de emergencia que ya fueron cargados previamente.
4. **Ofertas en Tiempo Real**: Como comprador, quiero recibir notificaciones push de ofertas flash de comercios a menos de 5km de mi ubicación.

## Comercio / Proveedor
1. **Gestión de Perfil**: Como dueño de un local, quiero actualizar mis fotos, horarios y descripción para atraer más clientes.
2. **Promociones**: Como carnicero, quiero publicar una oferta de "Asado" válida por 2 horas para liquidar stock sobrante.
3. **Conversión Directa**: Como proveedor, quiero un botón flotante de WhatsApp en mi perfil para que los clientes me contacten con un solo click.

## Administrador / Moderador
1. **Moderación de Reseñas**: Como moderador, quiero aprobar o rechazar comentarios para evitar lenguaje ofensivo o spam.
2. **Auditoría de Cambios**: Como administrador, quiero ver un historial de cambios realizados por los proveedores antes de que se publiquen (modo borrador/aprobado).
3. **Gestión de Usuarios**: Como super-admin, quiero poder suspender cuentas que incumplan las normas comunitarias.
4. **Dashboard de Sistema**: Como técnico, quiero ver el estado de uso de RAM y Disco de la RPi 5 desde el panel administrativo para prevenir caídas.
