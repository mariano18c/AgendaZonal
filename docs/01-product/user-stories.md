# Historias de Usuario Principales
1. **Búsqueda**: Como vecino, quiero buscar un plomero cerca de mi ubicación para resolver una urgencia.
2. **Confianza**: Como usuario, quiero ver reseñas verificadas para elegir un comercio seguro.
3. **Ofertas**: Como vecino, quiero recibir alertas de ofertas locales para ahorrar en mis compras diarias.
