# AgendaZonal: Roadmap 2026-2027

## Q2 2026: Consolidación y Premiumization
- [ ] **Premium Tiers**: Implementar lógica de niveles de visibilidad y badges de verificado.
- [ ] **UI Refactor**: Aplicar estética "Premium" (Glassmorphism + Animaciones suaves) en todo el frontend.
- [ ] **Push Engine**: Finalizar el sistema de notificaciones push para ofertas flash.
- [ ] **QA Hardening**: Alcanzar el 95% de cobertura en tests de integración.

## Q3 2026: Interactividad y Comunidad
- [ ] **Review Verification**: Sistema de validación de reseñas mediante fotos/tickets para evitar fraude.
- [ ] **Analytics Dashboard**: Panel para proveedores Premium con métricas detalladas de visitas.
- [ ] **Dark Mode Nativo**: Soporte completo para modo oscuro basado en preferencias de sistema.
- [ ] **Multi-zona**: Preparar la arquitectura para soportar múltiples municipios (Roldán, Funes) desde la misma instancia RPi 5.

## Q4 2026: Optimización y Escala
- [ ] **Image Proxy**: Generación dinámica de miniaturas WebP para reducir ancho de banda.
- [ ] **Background Workers**: Migración de tareas pesadas (envío de mails/notificaciones) a un proceso worker separado con RQ/Redis (lite).
- [ ] **Offline Sync**: Mejora de sincronización periódica en Background Sync API.

## Q1 2027: Ecosistema Extendido
- [ ] **Módulo Emergencias**: Sección especializada para Defensa Civil e instituciones públicas con datos en tiempo real.
- [ ] **API Pública**: Permitir que comercios locales integren sus propios sistemas de inventario simple.
- [ ] **Auto-Moderación IA**: Integración de modelos locales ligeros para pre-moderación de fotos y comentarios.
